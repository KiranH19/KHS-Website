# Karnataka High School Website

A simple and responsive school website built using **HTML, CSS, and JavaScript**.

## Project Description
This project is a frontend website for a high school in Karnataka. It provides basic information about the school, academics, facilities, admissions, and contact details.

## Tech Stack
HTML, CSS, JavaScript

## Features
- Homepage with school overview  
- About section  
- Academics / Courses section  
- Facilities section  
- Admission details  
- Contact section  
- Responsive design  
- Basic JavaScript interactions  

## Folder Structure
karnataka-highschool-website/
- index.html  
- style.css  
- script.js  
- images/  

## How to Run
1. Download or clone the repository  
2. Open index.html in any browser  


## Purpose
Built to practice frontend development and create a real-world school website project for portfolio.
